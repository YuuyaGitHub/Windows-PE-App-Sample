# Windows PE App Sample
This program is a sample app that runs only in Windows PE and is intended for educational and experimental purposes.
<br>It is very simple and simply displays `Hello, Windows PE!`.

Therefore, nothing will happen if you run it in normal Windows. It only runs in Windows PE.

> [!NOTE]
> The code for this program was generated by ChatGPT.

# Features
- CRT-free (no C runtime)
- Very small compiled size
- Windows PE only
- A simple example that simply displays a string on the console
  
# Build Procedure
1. Download this repository
2. Open Visual Studio 2022
3. Switch to the **x64 Developer Command Prompt** using the following command:
```
"%ProgramFiles%\Microsoft Visual Studio\2022\Community\VC\Auxiliary\Build\vcvars64.bat"
```
> [!WARNING]
> If you forget this, the application will be built as an x86 application, which will result in the following message when you run it in Windows PE: `This version of %1 is not compatible with the version of Windows you're running. Check your computer's system information and then contact the software publisher.`

4. Build with the following command:
```
cl /nologo /O1 /GS- winpe.c ^
/link /SUBSYSTEM:CONSOLE /ENTRY:mainCRTStartup /NODEFAULTLIB ^
kernel32.lib advapi32.lib
```

5. Run in Windows PE

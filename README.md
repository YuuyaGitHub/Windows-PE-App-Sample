# Windows PE App Sample
This is a sample application designed to run only on Windows PE and is primarily intended for educational and experimental purposes.
<br>It is a very simple program that simply outputs `Hello, Windows PE!`.

That's right, it's for Windows PE only and won't work on Normal Windows.

> [!NOTE]
> The code for this program was generated by ChatGPT.

# Features
- CRT-free (no C runtime)
- Very small compiled size
- Windows PE only
- A simple example that simply displays a string on the console
  
# Build Procedure
1. Download this repository
2. Open Visual Studio 2022
3. Switch to the **x64 Developer Command Prompt** using the following command:
```
"%ProgramFiles%\Microsoft Visual Studio\2022\Community\VC\Auxiliary\Build\vcvars64.bat"
```
> [!WARNING]
> If you forget to do this, when you run it in Windows PE you will get the this error: `This version of %1 is not compatible with the version of Windows you're running. Check your computer's system information and then contact the software publisher.`

4. Build with the following command:
```
cl /nologo /O1 /GS- winpe.c ^
/link /SUBSYSTEM:CONSOLE /ENTRY:mainCRTStartup /NODEFAULTLIB ^
kernel32.lib advapi32.lib
```

5. Run in Windows PE
